# 🐍 贪吃蛇游戏

一个使用纯HTML、CSS和JavaScript开发的经典贪吃蛇游戏。

## 🎮 游戏特性

- 🎨 精美的渐变UI设计
- ⌨️ 支持键盘方向键控制
- 🖱️ 支持鼠标点击按钮控制
- 📱 响应式设计，适配不同屏幕
- 💾 自动保存最高分记录
- ⏸️ 暂停/继续功能
- 🔄 重新开始功能

## 🚀 如何开始

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开始游戏"按钮或按方向键开始
3. 使用方向键或屏幕按钮控制蛇的移动
4. 吃到红色食物得分，避免撞墙或撞到自己

## 🎯 游戏规则

- 使用方向键（↑↓←→）或WASD键控制蛇的移动
- 也可以点击屏幕上的方向按钮
- 吃到食物得10分，蛇身会变长
- 撞到墙壁或自己的身体游戏结束
- 按空格键可以暂停/继续游戏

## 🛠️ 技术栈

- HTML5 Canvas
- CSS3 (渐变、动画、网格布局)
- 原生JavaScript (ES6+)
- LocalStorage (保存最高分)

## 🎨 特色功能

- **视觉效果**: 使用Canvas绘制，带有发光效果的蛇和食物
- **流畅控制**: 防止反向移动，确保游戏体验
- **得分系统**: 实时显示当前得分和历史最高分
- **游戏状态**: 支持开始、暂停、重置等多种状态
